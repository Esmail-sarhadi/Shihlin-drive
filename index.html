نقشه کامل رجیسترها (Register Map)

رجیسترهای پایه

آدرس نام فارسی نام انگلیسی محدوده دسترسی توضیحات
0000H-00C7H پارامترهای درایو Drive Parameters Pr0-Pr199 R/W دسترسی به تمام پارامترهای قابل تنظیم درایو
1000H حالت عملکرد Operation Mode 0000H-0008H R/W 0000H: حالت PU
0001H: حالت خارجی
0002H: حالت JOG
0003H-0008H: حالت‌های ترکیبی

رجیسترهای مانیتورینگ

آدرس نام فارسی نام انگلیسی محدوده دسترسی توضیحات

bit1: چرخش رو به جلو
bit2: چرخش معکوس
bit3: فرکانس رسیده

پارامترهای ارتباطی

پارامتر نام فارسی نام انگلیسی محدوده مقدار پیش‌فرض توضیحات
Pr170 انتخاب پروتکل Protocol Selection 0-1 0 0: پروتکل Shihlin
1: پروتکل MODBUS
Pr171 آدرس ارتباطی Communication Address 1-247 1 آدرس یکتای درایو در شبکه
Pr172 سرعت انتقال Baud Rate 0-2 1 0: 4800bps
1: 9600bps
2: 19200bps
Pr173 فرمت داده Data Format 0-5 0 ASCII Mode:
0: 7,N,2
1: 7,E,1
2: 7,O,1
RTU Mode:
3: 8,N,2
4: 8,E,1
5: 8,O,1

فرمت‌های پیام MODBUS

فرمت ASCII

فیلد طول محتوا توضیحات
STX 1 byte ':' کاراکتر شروع (3AH)
ADR 2 bytes 01-F7 آدرس درایو (1-247)
CMD 2 bytes کد دستور کد تابع MODBUS

فرمت RTU

فیلد طول محتوا توضیحات
START T1-T2-T3-T4 فاصله زمانی فاصله بیش از 10ms
ADR 1 byte 01H-F7H آدرس درایو (1-247)
CMD 1 byte 03H, 06H, 08H, 10H کد تابع
DATA n bytes N × 8 bits محتوای داده
CRC 2 bytes CRC16 چک‌سام CRC
END T1-T2-T3-T4 فاصله زمانی فاصله بیش از 10ms

کدهای خطا

کد خطا نام فارسی نام انگلیسی توضیحات راه‌حل
01H خطای کد تابع Function Code Error کد تابع نامعتبر است بررسی کد تابع درخواستی
02H خطای آدرس Address Error آدرس خارج از محدوده بررسی آدرس رجیستر
03H خطای مقدار داده Data Value Error مقدار داده نامعتبر بررسی محدوده مجاز داده
04H خطای دستگاه Device Error خطای پردازش درایو بررسی وضعیت درایو

مثال‌های کاربردی

مثال 1: خواندن پارامتر Pr0

درخواست (RTU):

01 03 00 00 00 01 84 0A
  • 01: آدرس درایو
  • 03: کد تابع (خواندن)
  • 0000: آدرس پارامتر
  • 0001: تعداد رجیستر
  • 840A: CRC

پاسخ (مقدار = 6%):

01 03 02 00 06 B9 0A
  • 01: آدرس درایو
  • 03: کد تابع
  • 02: تعداد بایت‌های داده
  • 0006: مقدار پارامتر (6)
  • B90A: CRC

مثال 2: تنظیم فرکانس 60Hz

درخواست (RTU):

01 06 10 02 17 70 F8 45
  • 01: آدرس درایو
  • 06: کد تابع (نوشتن تک رجیستر)
  • 1002: آدرس رجیستر فرکانس
  • 1770: مقدار (60Hz × 100)
  • F845: CRC

نکات پیکربندی و راه‌اندازی

پیکربندی اولیه

  1. تنظیم پروتکل (Pr170):
    • 0: پروتکل Shihlin
    • 1: پروتکل MODBUS
  2. تنظیم آدرس (Pr171):
    • محدوده: 1-247
    • هر درایو باید آدرس یکتا داشته باشد
  3. تنظیم سرعت ارتباط (Pr172):
    • 0: 4800bps
    • 1: 9600bps
    • 2: 19200bps
  4. تنظیم فرمت داده (Pr173):
    • ASCII Mode:
      • 0: 7,N,2
      • 1: 7,E,1
      • 2: 7,O,1
    • RTU Mode:
      • 3: 8,N,2
      • 4: 8,E,1
      • 5: 8,O,1

عیب‌یابی و نکات مهم

  • تنظیمات یکسان در دو طرف:
    • سرعت ارتباط
    • فرمت داده
    • پریتی
    • بیت‌های توقف
  • بررسی اتصالات فیزیکی:
    • اتصال صحیح A و B
    • شیلد کابل
    • مقاومت خاتمه خط
  • محاسبه CRC:
    • پلی‌نومیال: A001h
    • مقدار اولیه: FFFFh
    • بررسی صحت محاسبه

جزئیات رجیسترهای کنترلی و وضعیت

رجیستر کنترلی (1001H)

بیت نام توضیحات حالت‌ها
0 MRS توقف خروجی 1: توقف، 0: عادی
1 STF چرخش رو به جلو 1: فعال، 0: غیرفعال
2 STR چرخش معکوس 1: فعال، 0: غیرفعال
3 RL سرعت پایین 1: فعال، 0: غیرفعال
4 RM سرعت متوسط 1: فعال، 0: غیرفعال
5 RH سرعت بالا 1: فعال، 0: غیرفعال

رجیستر وضعیت (1001H)

بیت نام توضیحات حالت‌ها
0 RUN وضعیت عملکرد 1: در حال کار، 0: متوقف
1 FWD جهت چرخش 1: رو به جلو، 0: معکوس
2 SU رسیدن به فرکانس 1: رسیده، 0: نرسیده
3 OL اضافه بار 1: فعال، 0: عادی
7 ALM خطا 1: خطا، 0: عادی
